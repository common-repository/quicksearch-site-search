== Description ==

Automatically update the search index of QuickSearch.
You'll need an [QuickSearch Content Key, API key](http://www.quick-solution.com/quicksearch) to use it.

== Installation ==

Upload the QuickSearch plugin to your blog, Activate it, then enter your [QuickSearch Content Key, API key](http://www.quick-solution.com/quicksearch).
